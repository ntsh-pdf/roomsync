# RoomSync

RoomSync is a full-stack space booking and reservation system designed to simplify the management of meeting rooms, workspaces, and shared facilities. Users can view available spaces, create reservations, and manage bookings through a responsive and user-friendly interface.

## Features

- View available rooms and spaces
- Create and manage reservations
- Booking schedule management
- Responsive user interface
- RESTful API integration
- SQLite database storage

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- JavaScript

### Backend
- Node.js
- Express.js
- SQLite3
